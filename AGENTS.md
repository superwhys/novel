# 小说续写规则

本小说的正式书名为《我与篮球的距离》。

凡涉及本项目小说的续写、补章、扩写、改写或润色，必须遵守以下规则：

1. 开始写作前，完整阅读 `docs/story/WRITING_GUIDE.md`、`docs/story/STORY_BIBLE.md`、`docs/story/characters/README.md`、三位核心人物档案及郑天锐的人物档案，并以原稿 `docs/story/content/1.txt`、`docs/story/content/6.txt`、`docs/story/content/7.txt` 为语言风格的最高依据。
2. `docs/story/OUTLINE.md` 用于人物与剧情框架；后补的第 2—5 章只能作为连续性参考，不能取代原稿成为风格样本。
3. 故事主体固定在 2014 年的深圳光明新区公明中学。今天的马田街道当时尚未成立，正文采用 2014 年的行政称谓。
4. 主要学生角色统一为初三年级；已明确的背景和班级设定以 `docs/story/STORY_BIBLE.md` 为准。
5. 新增真实道路、学校、建筑、公交、商场、公园或其他地方信息前，必须查询其在目标时间是否存在及当时名称。优先使用政府、学校和交通等官方来源。
6. 保留原稿的少年热血、比赛速度、口语玩笑和直接情绪，但不要故意复制错别字、乱码或规则错误。
7. 使用 2014 年能够成立的语言、产品、网络文化、NBA 信息和交通条件，不得混入后来的地铁站、短视频梗或新建筑。
8. 新章节一旦明确了新的班级、人物关系、家庭、比赛结果或时间线事实，应同步更新 `docs/story/STORY_BIBLE.md`；属于三位核心人物或郑天锐的个人经历，还应更新 `docs/story/characters/` 中对应文件。
