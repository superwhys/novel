# 三位核心人物、重要辅助人物与兄弟群像

全书有三条并行的核心成长线：

- [杨浩文](杨浩文.md)：队长、担当与领导能力。
- [麦锐城](麦锐城.md)：自信外表、家庭压力与团队责任。
- [麦梓轩](麦梓轩.md)：天赋短板、感情挫折与篮球梦想。

三人可以理解为三位主角，拥有相近的成长分量。章节不必平均分配戏份，也不要求每章同时出现三人，但整部小说必须让三条人物线各自推进，并在班赛、家庭与学业压力、校队危机和区争霸赛中互相影响。

[郑天锐](郑天锐.md)是重要辅助人物。他保留超级第六人、关键球员和队友连接者的作用，可以有短篇幅的内心变化，但不与三位核心人物占用同等叙事篇幅。

[麦子洋](麦子洋.md)和[麦家俊](麦家俊.md)是七人兄弟群像中不可缺少的固定成员。麦子洋是稳重可靠的校队中锋和内线支柱，兄弟间称“洋哥”；他在第11—17章正常随队，已经从指出协防问题走到主动制定防守口令顺序，后续直接延续既有出场，不重新介绍。麦家俊是第15章以旧兄弟身份病愈返校的左撇子小前锋，兄弟间称“俊哥”；他的左手突破和分球感觉仍在，但连续跑动、回防体能和疲劳后的判断尚未恢复，第17章起开始接受受限训练和有限轮换，后续不按新人加入处理。两人不另设与三位核心人物同等篇幅的成长线，但必须拥有自己的判断、行动和兄弟互动，不能只作为名单或场上位置存在。

麦子洋的辅助变化是从“默默补位”走向“主动把内线看到的问题说清楚”；麦家俊的辅助变化是从“病后急着证明没有掉队”走向“接受恢复节奏，在有限任务中重新找到位置”。两条辅助线必须影响核心人物，但不替代麦梓轩、杨浩文和麦锐城完成主要成长。

七个兄弟为：麦梓轩、杨浩文、麦锐城、郑天锐、李永顺、麦子洋、麦家俊。

## 使用规则

- 每份人物档案只记录该人物已经明确的经历、关系、当前状态和仍未确定的事项。
- 新章节为三位核心人物、郑天锐、麦子洋或麦家俊增加经历后，应同步更新对应文件。
- 全局时间、比赛结果和共同关系仍记录在上一层的 `STORY_BIBLE.md`。
- 尚未在正文发生的大纲情节不能写成“既定经历”。
