# 《我与篮球的距离》续写风格与时代设定

本文件是后续续写、补章和修订时的长期创作规范。

人物、班级、比赛结果和时间线等连续性事实统一查阅 `STORY_BIBLE.md`；三位核心人物及郑天锐的个人经历查阅 `characters/`；本文件主要规定语言风格、时代口径和考据方法。

## 一、风格依据与优先级

判断语言和叙事风格时，按以下优先级执行：

1. 原稿第 1、6、7 章，即 `content/1.txt`、`content/6.txt`、`content/7.txt`。
2. `OUTLINE.md` 中的人物设定、小说大纲和章节框架。
3. 后续新增章节的既有剧情连续性。

第 2—5 章属于后来补写的内容，可以用于衔接剧情，但不能反过来取代第 1、6、7 章成为风格标准。原稿中偶尔存在错别字、标点混乱、视角误写和重复段落；后续要保留少年感和未经雕琢的冲劲，但不必故意模仿这些文字错误。

## 二、原作风格提取

### 1. 整体气质

这是一部带有明显少年创作气质的校园篮球小说。核心感觉是：热血、直接、幽默、自信，感情表达略带夸张，却是真诚的。

- 篮球不是背景，而是人物表达梦想、友情、自尊和失恋情绪的主要方式。
- 比赛段落追求“爽”和速度，日常段落依靠男生间的互损与玩笑放松节奏。
- 少年会把一次进球、一次单挑、一次挑衅看得很重。叙事应尊重这种当时当地的重量，不要站在成年人角度嘲笑他们。
- 情绪通常直接说出来，不采用过度含蓄、象征化或哲理化的表达。

第1—10章因承接原作未完成的比赛线，篮球内容较密集；这一比例不作为后续模板。从第11章开始，普通章节可大致理解为：三成篮球或训练、四成人物生活与成长、三成友情玩笑和青春烦恼。正式比赛章可以提高篮球比例，但比赛前后的章节必须把重心还给人物。比例不必机械执行，一章也不应长时间只有沉重情绪或纯粹比赛解说。

### 2. 叙事视角

- 采用第三人称叙事，按章节和场景在麦梓轩、杨浩文、麦锐城三条核心成长线之间切换；郑天锐作为重要辅助视角，只在关键比赛或确有叙事需要时短暂承接视角。前八章较多贴近麦梓轩，不代表全书是麦梓轩的个人成长故事。
- 视角距离较近，经常直接写人物心里的念头，例如“他不明白”“他心想”“他只想着”。
- 叙述者会直接评价人物和场面，如“十足地霸气”“十足的高富帅”“这下可摔得不轻”，不追求冷静客观。
- 不要频繁使用全知视角解释所有人的心理。同一小场景通常跟随一个主要人物。
- 不要求三位核心人物平均出现在每一章，但整部小说必须让三条成长线各自发展并互相影响，不能让杨浩文和麦锐城长期只充当麦梓轩的辅助角色。

### 3. 句子与段落节奏

- 以短句和短段落为主，一个动作、一次传球或一句话常单独成段。
- 比赛越紧张，句子越短，必要时可以使用独立的感叹句，如“快回防！”“隔扣！！！”。
- 可用“……”或空行完成快速转场，但不要每几段就堆叠分隔符。
- 景物描写只需两三笔，并应立刻服务于气氛或人物状态。例如灯光、夜空、晚风、球鞋摩擦地板的声音。
- 不写连续数百字的精致环境散文，也不要为了显得高级而大量使用复杂比喻。

### 4. 对话风格

- 对话占比要高。人物关系主要通过说话和行动体现，而不是由旁白概括。
- 男生之间习惯互损、吹牛、拆台，语言口语化，常有“卧槽”“坑爹”“吊打”“装逼”“弱爆了”等 2014 年前后常见表达。
- 杨浩文、麦锐城会自恋和吹牛；其他人常用“头上好多牛”“牛被吹上天了”一类话拆台。
- 玩笑之后仍要能看出队友间的信任。真正遇到受伤、失恋或低谷时，安慰应简短、笨拙但真诚。
- 教练说话直接，先指出具体问题，再给训练方法。不要让初三学生和教练突然说出过于成熟、像心理咨询文案一样完整的话。
- 不使用 2014 年以后才流行的网络词、短视频梗或饭圈表达。

### 5. 篮球比赛的写法

- 第11章以后不默认每章都有完整比赛。正式比赛应集中在大纲指定的赛事章；普通训练、单挑和队内对抗只写能够改变人物关系或验证成长的关键回合。
- 比赛场面必须结合真实篮球技战术来写。关键回合应交代基本站位、空间、掩护、换防、协防、轮转、卡位或传球路线，以及机会为什么出现，不能只罗列“胯下、后撤步、暴扣”等动作名称。可以合理使用挡拆顺下、弱侧空切、突分、联防、全场紧逼、破紧逼、快攻落位和内外线配合等战术，但复杂程度必须符合2014年公明中学初中校队或班级赛的执行能力；班赛普通学生不能突然打出职业球队级别的精密体系。
- 采用现场解说般的快速推进：持球人、对位者、动作、协防、结果、比分变化。
- 只详细写关键回合，不必逐个回合做流水账。
- 动作要具体：胯下、挡拆、顺下、急停、后撤步、欧洲步、空接、补防、打三分等。
- 可以使用英文术语后接中文解释，但只在有表现力时使用，例如“Euro step！欧洲步！”；不要把每个普通动作都翻译一遍。
- 重要进球允许夸张、重复和感叹号，以接近少年观看 NBA 集锦时的兴奋感。
- 比分、犯规、换人、命中率和位置等数据可以增强真实感，但必须前后一致。
- 球员不能只靠招式名称取胜。写清防守为何失位、掩护怎样创造空间、传球为什么出现机会。
- 角色球风应保持稳定：麦梓轩擅长速度、突破和传球，中远投是早期短板；郑天锐是关键时刻敢出手的第六人；麦锐城身体强壮、能扣篮且爱表现；麦子洋是稳定的内线支柱；李永顺善于跑位和判断；杨浩文自信、爱开玩笑，也愿意承担关键责任。

### 6. 青春感情线的写法

- 感情冲突与篮球相互影响，但不能取代篮球主线。
- 麦梓轩面对林知夏和麦浩鑫时，会受伤、愤怒、较劲，也会把这些情绪带进训练和比赛。
- 少年通常不擅长长篇分析自己的感受，更可能沉默、逞强、加练，或想在球场证明自己。
- 避免把女角色只写成奖品或反派。即使叙事贴近麦梓轩，也应让她们有自己的不满和选择。
- 不写成人化的亲密内容；感情尺度保持在初中校园范围内。

### 7. 人物成长与场外叙事

- 第11章以后，每章动笔前先写清楚这一章属于谁、他具体遇到什么生活问题、作出了什么选择。若答案只有“参加比赛并赢或输”，说明人物冲突仍不够完整。
- 家庭、学业和友情场景必须包含具体行动，不能只让人物坐着谈心。可以写赶作业、吃饭时的沉默、迟到后的撒谎、替兄弟收拾东西、答应了却没有出现、借笔记、被老师点名或回家后仍惦记训练。
- 父母、老师、教练和女性角色都应拥有合理立场，不能只作为阻挡主角梦想、发布任务或制造情绪的工具。
- 场外矛盾不能靠比赛胜利自动消失。比赛最多让人物看见问题；真正的变化需要坦白、道歉、守约、求助、安排时间或承担后果。
- 成长要允许反复。麦梓轩制定计划后可以先失败一次，麦锐城坦白后仍会下意识吹牛，杨浩文学会倾听后仍可能再次包办。阶段性进步不等于性格被彻底改好。
- 友情不只写在传球和庆祝里。六个兄弟可以在课堂、放学、家庭压力和失约中互相帮助，也会因为替人圆场、说话太重或隐瞒事实而发生冲突。
- 郑天锐、李永顺和麦家俊出场时应有自己的判断与选择，不能永远只负责接三分、空切、劝架或说笑话。配角主动做出的事应能改变至少一位核心人物的决定。
- 日常章节仍要保留少年感和动作感。人物可以一边互损一边帮忙，不需要把成长写成长篇心理分析或成人式谈心。

### 8. 章节结构与篇幅

一章通常可以采用以下节奏：

1. 用具体动作、突发消息或人物对话快速开场；不必默认从比赛和训练开始。
2. 提出一个明确的小冲突，例如成绩下滑、迟到失约、家庭分歧、朋友隐瞒、投篮失准或训练安排。
3. 通过人物行动和关系互动推进，而不是只靠旁白说明；篮球只是可选场景之一。
4. 让冲突出现阶段性变化，但不必每章彻底解决。
5. 用一个动作、没说出口的话、未完成的承诺、比分或新问题收尾，推动读者进入下一章。

篇幅以原稿为参照，通常约 2500—3500 个汉字。剧情需要时可以增减，不为了凑字数重复比赛过程。

### 9. 应避免的“非原作感”

- 过分华丽、文学化的长句和景物描写。
- 每次情绪变化都附带一段人生道理。
- 队友突然进行完整、专业的心理疏导。
- 所有人都说同一种整齐、礼貌、成熟的话。
- 只写抽象的“青春、梦想、成长”，却缺少拍球、跑位、打铁、汗水和具体行动。
- 把校园、家庭和友情写成两场比赛之间的过场，所有问题最终仍靠赢球解决。
- 连续多章重复“发现技术问题—加练—参加比赛—关键球证明自己”的结构。
- 过度干净的爽文：训练一次就解决短板，主角每个关键球都进，对手只会无脑挑衅。
- 故意复制原稿的错别字、乱码、重复段落和篮球规则错误。

## 三、固定时代与地点设定

### 1. 故事时间

- 故事主体发生在 **2014 年**。
- 默认季节和具体月份应由章节上下文决定；涉及真实 NBA 赛季、球鞋型号、考试、天气和节日时，先确定月份再写。
- 手机、社交和网络环境应符合 2014 年。可以出现 QQ、QQ 空间、短信、早期微信和贴吧；不要出现短视频直播、扫码点餐等后来的日常体验。
- 2014 年光明尚未通地铁。深圳地铁 6 号线在 2020 年才开通，因此不能让人物从红花山站、合水口站等地铁站出行。

### 2. 行政区划的年代口径

故事发生地的年代写法应为：

> 中国广东省深圳市光明新区，公明一带。

2014 年光明仍称“光明新区”，不能写成 2018 年才成立的“光明区”。当时光明新区主要为光明、公明两个办事处辖区；马田办事处在 2016 年 8 月才挂牌成立。

因此，正文中的正式地址或新闻口径应使用“光明新区公明办事处”或简写为“光明新区公明”。人物口语中可以只说“公明”。

用户指定的“马田街道”作为地理意图保留：人物的生活圈可以覆盖今天马田街道的合水口、薯田埔、马山头、根竹园等区域，但在 2014 年的叙事里不要把它们称作“马田街道”。需要说明今昔位置时，可在创作笔记中写“今马田街道”，不要让 2014 年的人物这样说。

### 3. 学校与周边锚点

- 故事核心学校是公明中学。
- 现实中的公明中学位于振明路 90 号，1976 年迁至红花山下；红花山公园位于振明路 88 号，紧邻学校，可用于放学、跑步、远眺、约谈等场景。
- 公明中心一带可使用振明路、望胜路、松白路、公明广场、公明汽车站等真实地名，但每次写入前仍要核对该地点在目标月份是否已存在、当时名称是什么。
- 2014 年校园应以改扩建前的旧校园为背景。不能直接套用 2018 年以后改扩建或 2024 年红花山体育中心升级后的建筑布局。
- 公明中学在小说中按初级中学设定，主要学生角色统一为初三年级。具体班级以 `STORY_BIBLE.md` 为准。

## 四、固定创作流程

以后每次续写、补章、扩写、改写、润色或进行其他正文创作，固定依次执行以下四步：

1. **读取整个框架**
   - 完整阅读 `WRITING_GUIDE.md`、`STORY_BIBLE.md`、`OUTLINE.md`、`characters/README.md`、三位核心人物档案及郑天锐人物档案。
   - 完整阅读原稿第 1、6、7 章，并阅读与本次剧情直接承接的相邻章节。
2. **列出大纲**
   - 正式写作前先提交详细大纲，至少包括核心人物、人物问题、场景顺序、关键行动或篮球回合、情绪变化、结尾钩子和连续性边界。
   - 等用户确认大纲后再开始写正文；若用户提出调整，先修改大纲。
3. **写文章**
   - 严格依据确认后的大纲写作；需要临时改变关键剧情、人物关系或比赛结果时，先说明原因并重新确认。
   - 正文完成后同步更新所有受影响的故事设定、大纲和人物档案。
4. **检查刚写内容的逻辑**
   - 写完后进行一次独立于写作过程的逻辑复查，不把“写完”视为“已经检查完”。
   - 逐项核对时间线、相邻章节事实、人物动机、对话指代、场景移动、物品来源、比赛比分与得分变化、篮球动作和战术因果、结尾伏笔。
   - 将发现的问题修正并再次检查后，才交付最终版本。

## 五、真实地点的查询规则

后续每次新增真实环境、建筑、学校、道路或交通方式时，先查询再写，流程如下：

1. 先确认剧情的具体年份和月份。
2. 优先查询深圳市政府、光明区政府、教育局、学校官网、交通部门、规划部门和当年的政府公报。
3. 同一地点区分“2014 年已经存在”“当时在建”“后来才建成”三种状态。
4. 当前地图只能用于确认空间关系，不能单独证明某建筑在 2014 年已经存在。
5. 若只有今天的资料，应使用保守描写，例如道路方向、红花山、湿热天气、公交出行等，不虚构具体店名和建筑外观。
6. 若剧情必须使用未经证实的地点，明确把名称设为虚构，不冒充真实史料。
7. 查询得到的关键事实和来源应补充到本文件，逐步形成小说自己的 2014 年公明资料库。

## 七、续写前检查清单

- 是否先读了第 1、6、7 章，而不是只模仿后补章节？
- 人物说话是否像 2014 年的深圳初三学生？
- 篮球动作、比分、犯规和球员位置是否合理且连续？
- 这一章是否同时有具体行动、人物互动和情绪变化？
- 这一章的核心矛盾是否在删掉比分后仍然成立？
- 若不是大纲指定的正式比赛章，是否把篮球场景控制在真正必要的关键片段？
- 家庭、学习或友情问题是否通过场外行动推进，而不是等待下一场球解决？
- 郑天锐、李永顺和麦家俊是否拥有自己的判断，而不只是服务核心人物？
- 是否保留了热血与玩笑，而没有写成沉重的成人文学？
- 新出现的真实地点是否查过 2014 年的存在状态和当时名称？
- 是否误用了“光明区”“马田街道”或 2020 年后的地铁设施？
- 结尾是否自然推动下一章？
